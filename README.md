# Production Python Bootstrapper Template

Bootstrapper is a reusable Python project template that establishes consistent engineering practices, repo organization, tooling, and development workflows. It provides developers with a clean starting point for backend services, machine learning projects, automation tools, and other Python applications while remaining agnostic to domain-specific technologies.

I'm building this bootstrap template to increate efficiency and consistency in my future projects,  
as well as to provide an easy start to other engineers who need a clean, consistent template for a new project.  
  
Creating a template will eliminate the mundane task of creating the same directories over and over again for each  
new project, ensure important folders are remembered, as well as enforce consistent engineering practices between projects.

Every project created with this template will include docker support, automated testing, consistent formatting,  
and clear documentation.
