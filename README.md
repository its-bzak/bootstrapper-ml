# ML Service Bootstrapper Template

This repository is a bootstrap template for ML Service projects, standardizing project structure, tooling, testing, containerization, and documentation.

I'm building this bootstrap template to increate efficiency and consistency in my future Machine Learning projects,  
as well as to provide an easy start to other engineers who need a clean, consistent template for a new project.  
  
Creating a template will eliminate the mundane task of creating the same directories over and over again for each  
new project, ensure important folders are remembered, as well as enforce consistent engineering practices between projects.

Every project created with this template will include docker support, automated testing, consistent formatting,  
and clear documentation.
